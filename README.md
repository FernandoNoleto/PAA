# PAA

## Como executar:

Para executar os algoritmos de ordenação primeiro execute o arquivo `Makefile`

```bash
$ make
```

Depois execute o arquivo `main`

```bash
$ ./main
```

##  Como usar:
Primeiro escolha o algoritmo de ordenação que deseja usar:
* 1 - Bubble sort
* 2 - Insertion sort
* 3 - Selection sort
* 4 - Merge sort
